# Network Troubleshooting Lab (Wireless Connection Issue)

## Overview
In this lab, I troubleshot a wireless network connectivity issue using Cisco Packet Tracer in a virtual environment.

## Problem
A device was unable to connect to a wireless network and had no internet access.

## Troubleshooting Steps
1. Verified wireless network (SSID) configuration
2. Checked IP configuration using `ipconfig`
3. Tested connectivity using `ping`
4. Identified incorrect or missing network settings
5. Reviewed wireless router configuration
6. Applied corrections and re-tested connectivity

## Tools Used
- Cisco Packet Tracer
- Command Prompt
- ipconfig
- ping

## Outcome
Successfully restored wireless connectivity. The lab was completed with a **100% completion rate**.

- Network troubleshooting
- Wireless networking
- IP configuration
